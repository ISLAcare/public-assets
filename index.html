GitHub access token

To upload or delete, paste a GitHub personal access token below. It's stored only in your browser's localStorage. Anyone with this token can push to the repo, so treat it like a password.

How to generate one
  1. Go to github.com/settings/personal-access-tokens/new
  2. Resource owner: ISLAcare
  3. Repository access: Only select repositories → public-assets
  4. Permissions → Repository permissions → Contents: Read and write
  5. Generate token, copy it, paste it below
  6. Approve the token in the ISLAcare org if prompted. If a classic token is easier, scope it to repo
